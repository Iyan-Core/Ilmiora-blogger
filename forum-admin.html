// Cara Publikasi ke Blog ILMIORA
01
Setelah klik Setujui — status thread berubah ke approved di Firebase. Artikel belum otomatis masuk ke Blogger (Blogger tidak punya API publik write).
02
Copy konten thread — klik ✎ Edit lalu copy judul + isi, buka Blogger → Postingan Baru → paste → tambah label kategori yang sesuai → Terbitkan.
03
Label Blogger yang cocok: Sejarah Islam / Kronik Umum / Teknologi / Biologi / dst. — sesuai kategori yang dipilih pengguna saat submit.
04
Artikel langsung muncul di mulai-membaca.html secara otomatis melalui Blogger API setelah terbit di blog — karena mulai-membaca sudah live-sync dari Blogger.
05
Tandai penulis — di Blogger tambahkan atribusi Ditulis oleh: [nama pengirim] di awal atau akhir artikel sebagai kredit kepada kontributor.
