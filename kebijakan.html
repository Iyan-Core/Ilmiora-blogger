01 //

Data yang Kami Kumpulkan

ILMIORA hanya mengumpulkan data yang benar-benar diperlukan untuk memberikan pengalaman terbaik kepada pengunjung. Data yang dikumpulkan meliputi:

  • Informasi akun (nama, email) yang kamu berikan saat mendaftar via Firebase Authentication
  • Data penggunaan anonim — halaman yang dikunjungi, durasi kunjungan (melalui Google Analytics)
  • Preferensi tampilan (tema gelap/terang) yang disimpan di browser lokal (localStorage)
  • Pesan yang kamu kirimkan melalui formulir kontak

Catatan penting: Kami tidak pernah menjual, menyewakan, atau memperdagangkan data pribadimu kepada pihak mana pun.

02 //

Cara Kami Menggunakan Data

Data yang terkumpul digunakan semata-mata untuk:

  • Menyediakan fitur akun dan personalisasi konten
  • Memahami konten apa yang paling bermanfaat bagi pembaca
  • Meningkatkan performa dan pengalaman platform
  • Merespons pertanyaan dan permintaan dari pengunjung
  • Mengirimkan notifikasi penting terkait platform (jika kamu berlangganan)
03 //

Cookie & Teknologi Pelacak

ILMIORA menggunakan cookie untuk meningkatkan pengalaman pengguna. Cookie adalah file teks kecil yang disimpan di browser. Jenis cookie yang kami gunakan:

  • Cookie Esensial — diperlukan untuk fungsi dasar platform (autentikasi, preferensi)
  • Cookie Analitik — membantu kami memahami pola kunjungan secara anonim (Google Analytics)
  • Cookie Fungsional — menyimpan preferensi seperti tema gelap/terang

Kamu dapat menonaktifkan cookie melalui pengaturan browser kapan saja, meskipun beberapa fitur platform mungkin tidak berfungsi optimal.

04 //

Layanan Pihak Ketiga

ILMIORA mengintegrasikan beberapa layanan pihak ketiga yang memiliki kebijakan privasi masing-masing:

  • Google (Firebase, Analytics, Blogger) — Autentikasi akun dan analitik pengunjung
  • GitHub Pages — Hosting file statis (PDF, MP3, halaman tambahan)
  • Islamic Network CDN — Streaming dan download audio murottal Al-Quran
  • Quran.com — Tautan baca Al-Quran online

Kami menyarankan kamu membaca kebijakan privasi masing-masing layanan tersebut untuk pemahaman yang lengkap.

05 //

Keamanan Data

Kami mengambil langkah-langkah teknis yang wajar untuk melindungi data pengunjung, termasuk:

  • Enkripsi data autentikasi melalui Firebase Authentication (standar industri)
  • Koneksi HTTPS di seluruh platform
  • Akses terbatas ke data pengguna hanya oleh admin terverifikasi

Meskipun demikian, tidak ada sistem yang 100% aman. Kami tidak dapat menjamin keamanan absolut dari transmisi data melalui internet.

06 //

Hak Pengunjung

Sebagai pengguna ILMIORA, kamu memiliki hak untuk:

  • Mengakses data pribadi yang kami simpan tentangmu
  • Meminta koreksi data yang tidak akurat
  • Meminta penghapusan akunmu dan data terkait
  • Menarik persetujuan penggunaan data kapan saja
  • Mengajukan keberatan atas pemrosesan data tertentu

Untuk menggunakan hak-hak ini, silakan hubungi kami melalui halaman Kontak.

Penggunaan platform secara berkelanjutan setelah perubahan diterbitkan dianggap sebagai persetujuanmu terhadap kebijakan yang diperbarui.
